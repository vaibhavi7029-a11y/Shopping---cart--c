Shopping Cart System in C

📌 Project Description

This project is a simple Shopping Cart System developed using C language.
It allows users to view products, add items to a cart, and generate a bill.

🚀 Features

- User Login System
- View Product List
- Add Products to Cart
- View Cart Items
- Generate Bill Receipt

🛠 Technologies Used

- C Programming Language
- Structures
- Functions
- Arrays

📂 Project Structure

shopping_cart.c
